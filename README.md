# f1-race-analytics
F1 Race Analytics Project
Overview

The F1 Race Analytics Project is a Python-based data analysis application that uses the FastF1 library to explore and visualize Formula 1 race data. It helps transform complex race telemetry into meaningful insights, allowing users to better understand driver performance, race strategies, and team decisions.

Features
Fastest lap detection
Driver performance comparison
Tyre performance analysis
Pit stop strategy insights
Sector-wise performance breakdown
Visualization of average lap times
Tech Stack
Python
FastF1
Pandas
NumPy
Matplotlib
Streamlit
