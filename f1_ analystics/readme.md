#  F1 Race Analytics Project

This project analyzes Formula 1 race data using the FastF1 library.

##  Features
- Fastest lap detection
- Driver performance comparison
- Tyre performance analysis
- Pit stop strategy insights
- Sector-wise performance
- Visualization of average lap times

##  Tech Stack
- Python
- FastF1
- Pandas
- NumPy
- Matplotlib

##  Output
Bar chart showing average lap time per driver.

## ▶ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the script:
   python main.py
